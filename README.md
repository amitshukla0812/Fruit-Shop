#  Fruit Shop - E-Commerce Web Application

A full-stack e-commerce web application for buying fresh fruits online.
Built with Java Spring Boot backend and Thymeleaf frontend.


##  Technologies Used

### Backend
- Java — Core programming language
- Spring Boot— Backend framework
- Spring Data JPA — Database operations & ORM
- REST API — API development
- MySQL — Relational database

### Frontend
- HTML — Structure
- CSS — Styling
- JavaScript — Dynamic functionality
- Thymeleaf — Server-side template engine
- BootStrap 


##  Features

### 👤 User Dashboard
- ✅ User Registration
- ✅ User Login / Logout
- ✅ Browse & Shop Products
- ✅ Add to Cart
- ✅ Place Order
- ✅ View Order History

### 🔧 Admin Dashboard
- ✅ Add New Products
- ✅ Update / Edit Products
- ✅ Delete / Remove Products
- ✅ Complete CRUD Operations
- ✅ View All Orders History
- ✅ View Registered Users
